# familyhub

A minimal in-memory Python package to manage a family's day to day coordination.

## Features
- **Finance** – track income and expenses and view the current balance.
- **Shared lists** – shopping, wish and todo lists with completion tracking.
- **Messaging** – send messages between family members.
- **Notifications** – store simple notifications per user.

## Development
Install `pytest` and run tests:

```bash
pytest
```
